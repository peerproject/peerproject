Velkommen til PeerProject fjern konsoll.

Gjennom denne web konsollen har du full kontroll over søk, nedlastning, uploads og nettverkskoblinger.


Søk - Søk etter filer, sjekk søke resultat, velg filer du vil laste ned.
Downloads - Kontroller nedlastningen, sett filer på pause, start opp filer og fjern ferdige.
Uploads - Vis alle uploads.
Nettverk - Sjekk og kontroller nettverkskoblingene, koble til eller koble i fra de forskjellige P2P nettverkene, eller skru av PeerProject.
