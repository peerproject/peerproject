PeerProject Compiling Notes:
___________________________


0. This sourcecode package is self-contained.  Only Visual Studio 2008 is required.


1. Install Microsoft Visual Studio 2008 + SP1
	- http://www.microsoft.com/downloads/details.aspx?FamilyID=83c3a1ec-ed72-4a79-8961-25635db0192b  (90day Trial)
	- http://www.microsoft.com/downloads/details.aspx?FamilyID=fbee1648-7106-44a7-9649-6d9f6d58056e  (Service Pack)
	- *Or use Visual C++ Express 2008.  VS2005/2010 files available but unsupported.

2. Install Windows 2008 SDK
	- http://www.microsoft.com/downloads/details.aspx?FamilyID=E6E1C3DF-A74F-4207-8586-711EBE331CDC
	- *Or just use default Vista SDK 6.0A - Note Win7 SDK is insufficient.

3. Locate this complete PeerProject sourcecode set.
	- That's it, there are no external dependencies or setup.

4. Open \VS2008\PeerProject.sln and click "Build Solution"
	- Expect some tweaking, see Build Log for troubleshooting.
